# ETL-Programs  
**Transforming extracted data using SQL queries & loading it onto a table or file**  
*(All programs written in Python, primarily focusing on ETL logic.)*

---

## üß© Introduction  
This project showcases how to **extract, transform, and load (ETL)** data using Python.  
It demonstrates working with structured data files (like CSV and JSON), applying SQL-style transformations, and loading the results into tables or output files.  

The aim is to simulate real-world data workflows ‚Äî from reading input data, performing cleansing, filtering, and aggregation, to applying business logic and writing processed outputs for reporting and analytics.

---

## üìÇ Repository Structure  

### üß† Programs  

#### **Airline_info.py**  
- Analyzes airline dataset.  
- Extracts and transforms flight and customer details.  
- Generates summarized insights on routes, timings, and performance.

#### **Banking_customers_info.py**  
- Processes banking customer data.  
- Performs segmentation and computes new financial metrics.  
- Loads cleaned data for further reporting.

#### **Dept_and_employee_info.py**  
- Works with departmental and employee records.  
- Demonstrates data joins, filters, and aggregate operations across datasets.

#### **Employee_info_check.py**  
- Performs data quality checks on employee datasets.  
- Flags missing, inconsistent, or invalid entries.

#### **Plan_info.py**  
- Analyses subscription/plan information for service-based datasets.  
- Evaluates usage, pricing, and revenue metrics.

#### **Telecom_info.py**  
- Tailored for telecom datasets.  
- Analyzes customer usage patterns, churn indicators, and service plan data.

*(Additional scripts and helper utilities are also included in the repository.)*
